CSS Units

What is css unit?

A CSS unit specifies the size of a property that you provide for an element or its content. For instance, if you wanted to change a paragraph's property margin, you would give it a certain value. This value must include CSS unite.

Units: Relative or Absolute

The two types of units absolute and relativeshould be taken into account while comparing all of the potential possibilities for which units to choose.

Absolute unit

Regardless of the parent element or window size, "absolute" units have the same size. Therefore, whether shown on a small phone screen or a huge display, a property set with an absolute unit value will appear the same size. When working on a project where responsiveness is not being taken into account, absolute units might be advantageous. For instance, desktop applications that cannot be scaled can be styled for the standard dimensions. You do not require content scaling if the screen is not.

unit Equivalent to
cm 1cm = 37.8px = 25.2/64in
mm 1mm = 1/10th of 1cm
in 1in = 2.54cm = 96px
Q 1Q = 1/40th of 1cm
pc 1pc = 1/6th of 1in
pt 1pt = 1/72nd of 1in
px 1px = 1/96th of 1in

Relative unit

Since relative units scale in accordance with the parent or window size, they are helpful for designing responsive websites.The table below lists some of the most helpful units for web development.

unit Relative to
em the Font size of the parent
rem Font size of the root element
ch The advance measure (width) of the glyph "0" of the element's font
ex x-height of the element's font
vw 1% of the viewport's width
vh 1% of the viewport's height
vmin 1% of the viewport's smaller dimension
vmax 1% of the viewport's larger dimension
